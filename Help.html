Animatiile lucreaza fie pe codul propriu-zis sau pe tabloul de sortare, care contine elementele unui vector si pozitia lor in acest vector.Astfel:

BubbleSort

Reprezinta elementul aflat pe pozitia i.
Reprezinta elementul aflat pe pozitia i+1 (cel cu care este comparat cel de pe pozitia i).
Reprezinta elementele care se afla deja pe pozitia lor finala (nu mai este nevoie de parcurgerea lor).

SelectionSort

Reprezinta elementul aflat pe pozitia i.
Reprezinta elementul cu valoare minima.
Reprezinta elementul aflat pe pozitia j (cel cu care este comparat elementul minim).
Reprezinta elementele care se afla deja pe pozitia lor finala (nu mai este nevoie de parcurgerea lor).

InsertionSort

Reprezinta elementul aflat pe pozitia j.
Reprezinta elementul aflat pe pozitia j-1 (cel cu care este comparat cel de pe pozitia j).

ShellSort

Reprezinta elementul aflat pe pozitia j.
Reprezinta elementul aflat pe pozitia i-p (cel cu care este comparat cel de pe pozitia j si unde p reprezinta pasul iteratiei curente a algoritmului).
