Quick Sort

void qSort(int vector[], int st, int m, int dr){

int temp, min, max, mij1;

//luam mijlocul intervalului

mij1 = vector[(st+dr)/2];

min= st; max = dr;

do{

while(vector[min] < mij1) min++;

while(vector[min] > mij1) max--;

//interschimbarea

if(min <= max){

temp = vector[min];

vector[min++] = vector[max];

vector[max--] = temp;

}

}while(min <= max);

//la fiecare pas sortam "mai bine" intervalul st-dr


//schimbarea intervalului

if(st < max)qSort(vector,st,max);

if(dr > min)qSort(vector,min,dr);

}

PREZENTAREA METODEI

                La fel ca si Mergesort, Quicksort este un algoritm bazat pe metoda Divide et Impera. Astfel Quicksort imparte mia intai tabloul in doua subtablouri mai mici: elementele mai mici si elementele mai mari. Quicksortul poate ordona apoi, in mod recursiv, subtablourile rezultate. Algroitmul urameaza urmatorii pasi:

                    1.Se alege un element dnumit pivot din vector (de obicei cel din mijloc ).

                    2.Partitionarea: rearanjarea elementelor astfel incat elementele mai mici decat pivotul sa vina inaintea lui , iar cele mai mari decat el dupa acesta (valorile egalese pot duce in oricare parte). Dupa partitionare pivotul se afla la pozitia sa finala.

                3.In mod recursiv se aplica pasii de mai sus subtabloui de elmente mai mici si separat celui cu elemente mai mari.

                O lista de 0 sau 1 elemnte este considerata sortata. Selectia pivotului si pasii partititiei se pot face in diferite moduri, alegerea unui mod anume inflentand eficienta algoritmului.

Timpuri de executare:

  1. Cazul mediu : O( N*log(N))
  2. Cazul defavorabil : O(N^2)
  3. Cazul cel mai bun: O( N*log(N))
  4. Memorie folosita : O( log(N))
  5. Stabil : DA
  6. Sortare descrescatoare: while(vector[min] > mijl) min++;

    while(vector[max] < mijl) max--;

  7. Sortare crescatoare: while(vector[min] < mijl) min++;

    while(vector[max] > mijl) max--;
